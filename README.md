# Personal Homepage Assignment

This repository contains a static personal homepage and a first technical blog post for the Algorithm Foundations 2026 first major assignment.

## Assignment Deliverables

- Personal homepage URL.
- Blog page reachable from the homepage.
- PDF report with links, screenshots, implementation process, and AI Agent usage notes.

## Project Structure

- `AGENTS.md`: instructions for AI Agent assisted development.
- `index.html`: homepage.
- `style.css`: shared page styles.
- `blog/first-blog.html`: first technical blog page.
- `docs/ai/decisions.md`: project decisions.
- `docs/ai/handoff.md`: development handoff notes.
- `report/report.tex`: LaTeX report draft.

## Report Build

Compile the report with XeLaTeX:

```powershell
cd report
xelatex report.tex
```

## Deployment Plan

Use GitHub Pages after the static site is complete.
